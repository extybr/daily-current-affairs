Инструкция по echo

Основные опции:
echo текст — вывести текст
echo -n — без перевода строки в конце
echo -e — интерпретировать escape-последовательности
echo -E — не интерпретировать escape-последовательности

Escape-последовательности (с -e):
\n — новая строка
\t — табуляция
\r — возврат каретки
\b — backspace
\a — звуковой сигнал (bell)
\\ — обратная косая черта
\0NNN — символ с восьмеричным кодом NNN
\xHH — символ с шестнадцатеричным кодом HH

Вывод переменных:
echo $PATH — вывод переменной окружения
echo "Текущий путь: $PWD" — текст с переменной

Цветной вывод:
echo -e "\033[31mКрасный текст\033[0m" — красный текст
echo -e "\033[1mЖирный текст\033[0m" — жирный текст

Специальные файлы:
echo текст > файл — записать в файл (перезаписать)
echo текст >> файл — добавить в конец файла

Массивы:
echo ${array[@]} — все элементы массива

Примеры:
echo "Hello World" — просто вывод
echo -n "Введите имя: " — без перевода строки
echo -e "Строка 1\nСтрока 2" — две строки
echo -e "Колонка1\tКолонка2" — с табуляцией
echo -e "Звук\a" — со звуковым сигналом
echo -e "Текст\b\b" — с backspace
echo "Домашний каталог: $HOME" — с переменной
echo * — список файлов (подстановка shell)
echo {1..10} — последовательность чисел
echo $((2 + 2)) — арифметическое выражение

Перенаправление:
echo "log message" >> app.log — добавление в лог
echo "config=value" > config.ini — создание конфига

Многострочный вывод:
    echo "Первая строка
    Вторая строка
    Третья строка"
    
Escape-символы без -e:
echo "Текст\nКак есть" — выведет \n как текст

Проверка вывода:
echo $? — код возврата последней команды

Генерация строк:
echo {A..Z} — буквы A-Z
echo {a..z} — буквы a-z

Форматированный вывод:
printf — для сложного форматирования (лучше чем echo)

Встроенная команда vs /bin/echo:
type echo — показывает является ли встроенной командой
Встроенный echo может отличаться от /bin/echo
