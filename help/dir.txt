овные опции:
dir — список файлов (аналогично ls -C)
dir -a — все файлы (включая скрытые)
dir -l — подробный список
dir -1 — по одному файлу на строку

Сортировка:
dir -t — сортировка по времени изменения
dir -r — обратный порядок сортировки
dir -S — сортировка по размеру
dir -X — сортировка по расширению
dir -U — без сортировки

Форматы вывода:
dir -m — через запятую
dir -x — по горизонтали
dir -C — в несколько колонок (по умолчанию)

Классификация:
dir -F — добавить индикаторы типов (*/=>@|)
dir --color — цветной вывод
dir --color=auto — цветной только при терминале

Время:
dir -c — сортировка по ctime
dir -u — сортировка по atime

Рекурсивно:
dir -R — рекурсивный обход подкаталогов

Права и владельцы:
dir -n — числовые UID/GID вместо имен
dir -o — как -l, но без группы
dir -g — как -l, но без владельца

Примеры:
dir — простой список в колонках
dir -la — подробно со скрытыми файлами
dir -l --color=always — цветной вывод всегда
dir *.txt — только txt файлы
dir -d */ — только подкаталоги
dir -lt — отсортировано по времени (новые сверху)
dir -ltr — отсортировано по времени (старые сверху)
dir -lS — отсортировано по размеру (большие сверху)
dir -lSh — размеры в человеко-читаемом формате
dir -R /etc/ — рекурсивно каталог /etc

Размеры:
dir -lh — человеко-читаемые размеры (K, M, G)
dir --block-size=M — размеры в мегабайтах

Фильтрация:
dir -p | grep /$ — только каталоги
dir -l | grep "^-" — только обычные файлы
dir -l | grep "^l" — только симлинки

Форматирование времени:
dir -l --time-style=full-iso — полный ISO формат времени
dir -l --time-style=long-iso — длинный ISO формат

Группировка:
dir --group-directories-first — каталоги в начале

Специальные опции:
dir -A — все файлы кроме . и ..
dir -B — игнорировать файлы с ~ в конце
dir -I шаблон — исключить по шаблону
dir -N — не обрабатывать escape-последовательности

Примеры комбинаций:
dir -lath — все файлы подробно с человеко-читаемыми размерами
dir -lSh --color=auto — цветная сортировка по размеру

Малоизвестные опции:
dir -b — непечатные символы как \xxx
dir -q — заменять непечатные символы на ?
dir -i — показать inode номер
dir -Z — контекст SELinux

Важно:
В Linux dir обычно это просто алиас для ls -C
Проверить: type dir или alias dir

Отличия от ls:
dir по умолчанию выводит в колонках (ls -C)
ls по умолчанию выводит вертикально в одну колонку
