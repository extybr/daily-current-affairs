Инструкция по more

Основные команды:
more файл — открыть файл для просмотра
команда | more — просмотреть вывод команды
SPACE или f — следующая страница
ENTER — следующая строка
b — предыдущая страница
q или Q — выход

Поиск:
/паттерн — поиск паттерна вперед
n — следующее совпадение
' — вернуться к позиции перед поиском

Навигация:
= — показать номер текущей строки
:f — показать имя файла и номер строки
v — открыть текущий файл в редакторе
!команда — выполнить shell команду

Переход к строке:
more +номер файл — начать с указанной строки
more +/паттерн файл — начать с первого вхождения паттерна

Примеры:
more /etc/passwd — просмотреть файл паролей
ls -la | more — просмотреть список файлов
dmesg | more — просмотреть вывод ядра
more +50 log.txt — начать с 50-й строки
more +/error system.log — найти первое "error"
more -d file.txt — с подсказками (нажмите [h] для справки)
more -c file.txt — отображать с очисткой экрана
more -p pattern file.txt — начать с паттерна (аналог +/)
more -s file.txt — сжимать несколько пустых строк в одну

Множественные файлы:
more файл1 файл2 — просмотреть несколько файлов
:n — следующий файл
:p — предыдущий файл

Управление выводом:
-l — не останавливаться на символе ^L (form feed)
-u — не использовать подчеркивания и обратный видеорежим

Интерактивные команды:
. — повторить предыдущую команду
h или ? — показать справку
d или Ctrl+D — прокрутить на 11 строк
z — прокрутить на один экран (можно указать число: 5z)
s — пропустить вперед на одну строку
Ctrl+L — перерисовать экран

Пользовательские настройки:
-число — установить размер окна в строках
more -10 file.txt — показывать по 10 строк за раз

Переменные окружения:
MORE — опции по умолчанию
LESS — более продвинутый аналог more

Примеры использования:
cat большой_файл.txt | more — просмотр больших файлов
find / -name "*.conf" | more — просмотр найденных файлов
ps aux | more — просмотр процессов
netstat -tulpn | more — просмотр сетевых соединений
history | more — просмотр истории команд

Просмотр с выводом номера строки:
more -d +100 file.txt — начать со строки 100 с подсказками

Фильтрация:
grep pattern file.txt | more — поиск и постраничный вывод

Отличие от less:
more — только вперед, проще
less — вперед и назад, больше функций

Сжатие пустых строк:
more -s — полезно для логов с множеством пустых строк

Просмотр бинарных файлов:
more может показывать бинарные файлы, но лучше использовать hexdump или xxd

Режим редактора:
v — открывает файл в редакторе, указанном в $EDITOR

Справка внутри more:
Нажмите h во время просмотра для краткой справки

Автоматический выход:
Если файл меньше одного экрана — more покажет его и выйдет

Сохранение позиции:
Некоторые версии more поддерживают сохранение позиции при выходе

Совместимость:
more — стандартная POSIX утилита, есть во всех Unix-системах

Ограничения:
    Нельзя прокручивать назад (используйте less или PgUp)
    Меньше функций поиска чем в less
    Не поддерживает цветной вывод напрямую

Полезные комбинации:
more +$(grep -n "начало" file.txt | cut -d: -f1) file.txt — начать с найденной строки
