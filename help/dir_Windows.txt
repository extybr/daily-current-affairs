Инструкция по dir (Windows)

Основные команды:
dir — список файлов и папок
dir /p — постраничный вывод
dir /w — широкий формат (без деталей)
dir /b — только имена (bare format)
dir /s — рекурсивно все подкаталоги
dir /a — все файлы включая скрытые и системные

Атрибуты файлов:
dir /a:d — только директории
dir /a:-d — только файлы (не директории)
dir /a:h — скрытые файлы
dir /a:-h — не скрытые файлы
dir /a:s — системные файлы
dir /a:r — файлы только для чтения
dir /a:a — файлы готовые для архивации
dir /a:-r — файлы не только для чтения

Комбинации атрибутов:
dir /a:dh — директории и скрытые
dir /a:hs — скрытые и системные
dir /a:r-h — только для чтения, но не скрытые

Сортировка:
dir /o:n — по имени (алфавитно)
dir /o:-n — по имени (обратный порядок)
dir /o:e — по расширению
dir /o:s — по размеру (маленькие первыми)
dir /o:-s — по размеру (большие первыми)
dir /o:d — по дате (старые первыми)
dir /o:-d — по дате (новые первыми)
dir /o:g — сначала директории

Форматы времени:
dir /t:c — время создания
dir /t:a — время последнего доступа
dir /t:w — время последней записи (по умолчанию)

Поиск:
dir *.txt — все txt файлы
dir /s *.exe — все exe файлы рекурсивно
dir /s /b *.dll — только имена всех dll файлов
dir test*.* — файлы начинающиеся на "test"
dir /a-d *.* — только файлы (без папок)

Фильтрация:
dir | find "txt" — найти строки с "txt"
dir | findstr /i "text" — регистронезависимый поиск
dir | more — постраничный вывод через more

Информация:
dir /q — показать владельца файлов
dir /c — отображать разделители тысяч в размере файлов
dir /l — имена в нижнем регистре
dir /n — длинный список (как в Unix)
dir /x — показать короткие имена (8.3)
dir /-c — не показывать разделители тысяч

Примеры:
dir /s /b C:\Windows\*.exe — все exe файлы в Windows с путями
dir /od — отсортировать по дате (старые первыми)
dir /o-s — отсортировать по размеру (большие первыми)
dir /a-d /o-s — только файлы, отсортированные по размеру
dir *.doc /s /p — все doc файлы с паузами
dir /a:h C:\Users\Username\* — скрытые файлы в профиле
dir /t:c /od — отсортировать по времени создания
dir /q /a — все файлы с владельцами
dir /x /p — показать короткие имена с паузами

Специальные пути:
dir %TEMP% — содержимое временной папки
dir "%USERPROFILE%\Desktop" — рабочий стол
dir "C:\Program Files" — кавычки для путей с пробелами

Фильтры по дате:
dir /od | find "2024" — файлы 2024 года

Объём информации:
dir /s > dirlist.txt — сохранить полный список в файл
dir /s | find /c /v "" — подсчитать количество файлов

Дополнительно:
tree — древовидная структура каталогов
where — найти файлы (аналог which в Unix)

PowerShell альтернативы:
Get-ChildItem — более мощный аналог dir
ls — алиас для Get-ChildItem в PowerShell
gci — сокращение для Get-ChildItem
