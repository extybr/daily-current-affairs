Инструкция по grep

Основные опции:
grep -i — игнорировать регистр
grep -v — инвертировать поиск (исключить строки)
grep -n — показать номера строк
grep -c — подсчитать количество совпадений
grep -l — показать только имена файлов с совпадениями
grep -L — показать имена файлов без совпадений

Поиск:
grep -w — искать целые слова
grep -x — искать целые строки
grep -A N — показать N строк после совпадения
grep -B N — показать N строк до совпадения
grep -C N — показать N строк вокруг совпадения

Файлы:
grep -r — рекурсивный поиск в директориях
grep -R — рекурсивный поиск (+ симлинки)
grep -f FILE — читать шаблоны из файла
grep --include=*.ext — искать только в файлах с расширением
grep --exclude=*.ext — исключить файлы с расширением

Вывод:
grep -o — показать только совпавшую часть
grep -q — тихий режим (только код возврата)
grep -s — подавить сообщения об ошибках
grep -H — всегда показывать имя файла
grep -h — никогда не показывать имя файла

Цвет:
grep --color=always — всегда раскрашивать вывод
grep --color=auto — раскрашивать при выводе в терминал
grep --color=never — не раскрашивать

Контекст управления:
grep -m N — остановиться после N совпадений
grep -b — показать смещение в байтах
grep --binary-files=TYPE — обработка бинарных файлов

Синтаксис:
grep -E — использовать расширенные regexp (как egrep)
grep -F — искать как строки (как fgrep)
grep -G — использовать базовые regexp (по умолчанию)
grep -P — использовать Perl-совместимые regexp
