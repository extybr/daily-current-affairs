Инструкция по ls

Основные опции:
ls — список файлов
ls -l — подробный список
ls -a — показать все файлы (включая скрытые)
ls -h — человеко-читаемые размеры
ls -t — сортировка по времени изменения
ls -r — обратный порядок сортировки
ls -S — сортировка по размеру

Форматы вывода:
ls -1 — по одному файлу на строку
ls -m — через запятую
ls -C — в несколько колонок (по умолчанию)
ls -x — по горизонтали

Классификация:
ls -F — добавить индикаторы типов (*/=>@|)
ls --color — цветной вывод
ls --color=auto — цветной только при терминале

Сортировка:
ls -X — сортировка по расширению
ls -U — без сортировки (порядок в каталоге)
ls -v — естественная сортировка чисел

Время:
ls -c — сортировка по времени изменения статуса (ctime)
ls -u — сортировка по времени доступа (atime)

Права:
ls -Z — контекст SELinux
ls -n — числовые UID/GID вместо имен

Рекурсивно:
ls -R — рекурсивный обход подкаталогов

Показ ссылок:
ls -i — показать inode номер
ls -L — следовать симлинкам
ls -H — следовать симлинкам в командной строке

Примеры:
ls — простой список
ls -la — подробно со скрытыми файлами
ls -lh — размеры в КБ, МБ, ГБ
ls -lt — отсортировано по времени (новые сверху)
ls -ltr — отсортировано по времени (старые сверху)
ls -lS — отсортировано по размеру (большие сверху)
ls -lSh — размеры + сортировка по размеру
ls -la --color=always — цветной вывод всегда
ls -d */ — только подкаталоги
ls *.txt — только txt файлы
ls -la ~/ — домашний каталог подробно
ls -R /etc/ — рекурсивно каталог /etc
ls -i file.txt — inode файла
ls -l --time=atime — показать время доступа
ls -l --time=ctime — показать время изменения статуса
ls -l --block-size=M — размеры в мегабайтах

Фильтрация:
ls -p | grep / — только каталоги
ls -l | grep "^-" — только обычные файлы
ls -l | grep "^l" — только симлинки

Форматирование времени:
ls -l --time-style=full-iso — полный ISO формат времени
ls -l --time-style=long-iso — длинный ISO формат
ls -l --time-style=locale — локальный формат

Группировка:
ls --group-directories-first — каталоги в начале

Специальные опции:
ls -A — все файлы кроме . и ..
ls -B — игнорировать файлы с ~ в конце (резервные)
ls -I шаблон — исключить по шаблону
ls -N — не обрабатывать escape-последовательности

Примеры комбинаций:
ls -alth — все файлы подробно с размерами отсортированные по времени
ls -lSh --color=auto — цветная сортировка по размеру
