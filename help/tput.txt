Инструкция по tput

Цвета:
tput setaf N — цвет текста (0-7)
tput setab N — цвет фона (0-7)
*Цвета: 0-чёрный, 1-красный, 2-зелёный, 3-жёлтый, 4-синий, 5-пурпурный, 6-голубой, 7-белый*

Атрибуты текста:
tput bold — жирный
tput dim — тусклый
tput smul/rmul — подчёркивание вкл/выкл
tput rev — инвертировать цвета
tput smso/rmso — выделенный режим вкл/выкл
tput sgr0 — сбросить все атрибуты

Курсор:
tput cup Y X — переместить курсор (строка Y, колонка X)
tput sc/rc — сохранить/восстановить позицию
tput cnorm/civis — показать/скрыть курсор

Очистка:
tput clear — очистить экран
tput el — очистить до конца строки
tput el1 — очистить от начала до курсора
tput ed — очистить до конца экрана

Размеры:
tput lines — количество строк
tput cols — количество колонок

Клавиши:
tput khome — код Home
tput kend — код End
tput kdch1 — код Delete
