Инструкция по wc

Основные опции:
wc файл — количество строк, слов, байт в файле
wc -l — только количество строк
wc -w — только количество слов
wc -c — только количество байт
wc -m — количество символов
wc -L — длина самой длинной строки

Комбинации:
wc -lw — строки и слова
wc -lc — строки и байты
wc -wc — слова и байты
wc -lwm — строки, слова, символы

Несколько файлов:
wc файл1 файл2 — статистика по каждому файлу и сумма
wc *.txt — все txt файлы
wc -l *.py — строки во всех python файлах

Чтение из stdin:
команда | wc -l — подсчитать вывод команды
wc -l < файл — подсчитать строки в файле (только вывод счетчика)

Примеры:
wc document.txt — строки, слова, байты в файле
wc -l access.log — количество строк в лог-файле
wc -w essay.txt — количество слов в сочинении
wc -c image.jpg — размер файла в байтах
wc -m текст.txt — количество символов (важно для Unicode)
wc -L script.py — длина самой длинной строки в питоновском скрипте
wc -lw config.conf — строки и слова в конфиге
wc *.c — статистика по всем C файлам
wc -l *.log | tail -1 — общее количество строк во всех логах
find . -name "*.js" -exec wc -l {} + — строки во всех JS файлах
cat файл | wc -l — количество строк через pipe
echo "hello world" | wc -w — количество слов в строке (2)
echo -n "test" | wc -c — количество байт без перевода строки (4)
printf "тест" | wc -c — байт в UTF-8 строке (8)
printf "тест" | wc -m — символов в строке (4)

Различия -c и -m:
Для ASCII: -c и -m одинаковы
Для UTF-8: -c — байты, -m — символы (графемы)

Пример с Unicode:
    echo "café" | wc -c  # 5 байт (c a f é \n)
    echo "café" | wc -m  # 4 символа + \n = 5

Самые длинные строки:
wc -L file.txt — показывает длину самой длинной строки
wc --max-line-length — то же что -L

Итого:
При работе с несколькими файлами последняя строка — суммарная статистика

Практическое применение:
wc -l /etc/passwd — количество пользователей в системе
wc -l /var/log/auth.log — количество записей в логе аутентификации
wc -w document.txt — проверка объема текста
wc -c < large_file.iso — размер файла без имени файла в выводе
ls -la | wc -l — количество файлов в директории (включая . и ..)
find . -type f | wc -l — количество файлов рекурсивно
git diff | wc -l — количество измененных строк в git

Проверка кода:
wc -l *.php — строки кода в PHP файлах
wc -l --total=never *.py — только по файлам, без итога

Флаги управления выводом:
--files0-from=F — читать имена файлов из F (разделены нулем)
--total=auto — показывать итог если больше одного файла (по умолчанию)
--total=always — всегда показывать итог
--total=never — никогда не показывать итог

Пример с --total:
wc -l --total=never *.txt — статистика по файлам без итоговой строки

Подсчет слов:
Слово — это последовательность непробельных символов
echo " hello world " | wc -w — 2 слова (лишние пробелы игнорируются)

Бинарные файлы:
wc -c binary.bin — размер бинарного файла в байтах

Производительность:
wc работает очень быстро даже с большими файлами

Справка и версия:
wc --version — версия
wc --help — справка

Комбинирование с сортировкой:
wc -l *.log | sort -n — отсортировать файлы по количеству строк
wc -l *.py | grep -v total | sort -nr | head -10 — топ-10 файлов по количеству строк

Псевдонимы:
alias countlines='wc -l' — алиас для подсчета строк
alias countwords='wc -w' — алиас для подсчета слов
