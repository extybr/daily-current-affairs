Инструкция по sort

Основные опции:
sort file — отсортировать строки
sort -r — обратный порядок (descending)
sort -n — числовая сортировка
sort -h — сортировка человеко-читаемых чисел (2K, 1G)
sort -V — версионная сортировка (1.2, 1.10)

Сортировка по полям:
sort -k N — сортировать по полю N
sort -k N,M — сортировать по полям с N по M
sort -t ':' -k 3 — сортировать по 3-му полю с разделителем :

Типы данных:
sort -g — общая числовая сортировка
sort -M — сортировка месяцев (JAN, FEB...)

Управление дубликатами:
sort -u — удалить дубликаты
sort --unique — то же что -u

Стабильность:
sort -s — стабильная сортировка (сохранять порядок равных элементов)

Регистр:
sort -f — игнорировать регистр
sort --ignore-case — то же что -f

Проверка:
sort -c — проверить отсортирован ли файл
sort -C — проверить без вывода

Разделители:
sort -t ',' — разделитель запятая
sort -t $'\t' — разделитель табуляция

Порядок сортировки:
sort --reverse — обратный порядок
sort --dictionary-order — словарный порядок

Параметры полей:
sort -k 2,2 — сортировать только по второму полю
sort -k 2n — сортировать второе поле как число
sort -k 2,2 -k 3,3 — сортировать по полю 2, затем по полю 3

Примеры:
sort file.txt — отсортировать строки в файле
sort -r file.txt — обратная сортировка
sort -n file.txt — числовая сортировка
sort -u file.txt — удалить дубликаты строк
sort -t ':' -k 3 -n /etc/passwd — сортировать по UID
sort -k 2,2 -k 1,1 file.csv — сортировать по полю 2, затем по полю 1
sort -k 1.2,1.2 file.txt — сортировать по второму символу первого поля
sort -h sizes.txt — сортировать размеры (1K, 2M, 3G)
sort -V versions.txt — версионная сортировка
sort -M months.txt — сортировать по месяцам
sort -f mixed_case.txt — игнорировать регистр
sort -c sorted.txt — проверить отсортирован ли файл
sort -s -k 2 file.txt — стабильная сортировка по второму полю

Специальные опции:
sort --parallel=N — использовать N процессоров
sort --buffer-size=SIZE — размер буфера
sort --output=FILE — вывод в файл

Потоковая обработка:
sort - — читать из stdin
ls -la | sort -k 5 -n — отсортировать файлы по размеру

Локализация:
sort --dictionary-order — учитывать локаль
LC_ALL=C sort — быстрая сортировка без локализации

Слияние:
sort -m file1 file2 — слияние уже отсортированных файлов

Проверка:
sort -c — проверка без сортировки
sort --check — то же что -c
