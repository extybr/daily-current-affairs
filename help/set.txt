Чистая инструкция по основным опциям set:

set -e — немедленный выход при ошибке любой команды
set +e — разрешить продолжение после ошибок (режим по умолчанию)

set -u — ошибка при использовании неопределённых переменных
set +u — разрешить неопределённые переменные

set -x — печатать каждую команду перед выполнением (отладка)
set +x — отключить печать команд

set -f — отключить подстановку файловых шаблонов (*, ?, [])
set +f — включить подстановку шаблонов

set -H — включить history expansion с восклицательными знаками
set +H — отключить history expansion

set -C — запретить перезапись существующих файлов перенаправлением
set +C — разрешить перезапись файлов

set -o pipefail — считать пайп неудачным при ошибке любой команды в цепочке
set +o pipefail — пайп успешен если последняя команда успешна

set -o noclobber — то же что -C
set -o errexit — то же что -e
set -o nounset — то же что -u
set -o xtrace — то же что -x

set -- — установить позиционные параметры
set - — установить флаги из аргументов
