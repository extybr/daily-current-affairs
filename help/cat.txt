Инструкция по cat

Основные опции:
cat файл — вывести содержимое файла
cat файл1 файл2 — объединить несколько файлов
cat -n — добавить номера строк
cat -b — добавить номера только непустым строкам
cat -s — сжать несколько пустых строк в одну
cat -E — показать символы конца строки ($)
cat -T — показать табуляции как ^I
cat -A — показать все непечатные символы (-vET)
cat -v — показать непечатные символы

Перенаправление:
cat > файл — создать файл (ввод с клавиатуры, Ctrl+D)
cat >> файл — добавить в конец файла
cat файл1 > файл2 — копирование файла
cat файл1 файл2 > файл3 — объединение в новый файл

Потоковая обработка:
cat - — читать из stdin
команда | cat -n — добавить номера строк к выводу команды

Примеры:
cat document.txt — показать файл
cat -n script.sh — показать с номерами строк
cat file1.txt file2.txt — показать оба файла подряд
cat > newfile.txt — создать новый файл с вводом
cat >> log.txt — добавить текст в конец лога
cat -A config.conf — показать все символы включая непечатные
cat -s data.txt — убрать повторяющиеся пустые строки
cat -E script.py — показать где концы строк

Создание файлов:
    cat > list.txt << EOF
    item1
    item2
    item3
    EOF

Объединение:
cat part1 part2 part3 > full_document — собрать файл из частей

Просмотр бинарных файлов:
cat -v binary.bin — просмотр бинарного файла

Нумерация:
cat -n — нумерация всех строк
cat -b — нумерация только непустых строк

Фильтрация:
cat file.txt | grep pattern — поиск в файле
cat file.txt | head -20 — первые 20 строк

Кодировки:
cat file.txt | iconv -f CP1251 -t UTF-8 — конвертация кодировки

Конкатенация с буферизацией:
cat file1 file2 | less — просмотр через less

Сжатие пустых строк:
cat -s — полезно для логов с множеством пустых строк

Отображение табуляций:
cat -T — визуализация табов в коде

Проверка конца строк:
cat -E — увидеть Windows (\r\n) vs Unix (\n) переводы строк

Создание нескольких файлов:
    for i in {1..5}; do cat > "file$i.txt" <<< "Content $i"; done
