Инструкция по tr

Основной синтаксис:
tr SET1 SET2 — заменить символы из SET1 на SET2
tr -d SET1 — удалить символы из SET1
tr -s SET1 — сжать повторяющиеся символы из SET1

Наборы символов:
a-z — все строчные буквы
A-Z — все заглавные буквы
0-9 — цифры
[:alpha:] — буквы
[:alnum:] — буквы и цифры
[:digit:] — цифры
[:lower:] — строчные буквы
[:upper:] — заглавные буквы
[:space:] — пробельные символы
[:blank:] — пробел и табуляция
[:punct:] — знаки пунктуации
[:cntrl:] — управляющие символы
[:print:] — печатные символы
[:graph:] — печатные символы кроме пробела
[:xdigit:] — шестнадцатеричные цифры

Опции:
tr -c SET1 SET2 — дополнить SET1 (работать с символами НЕ из SET1)
tr -t SET1 SET2 — обрезать SET1 до длины SET2
tr -C SET1 SET2 — то же что -c но с наборами символов

Преобразования:
tr 'a-z' 'A-Z' — преобразовать в верхний регистр
tr 'A-Z' 'a-z' — преобразовать в нижний регистр
tr -d '\n' — удалить переводы строк
tr -d '[:punct:]' — удалить пунктуацию
tr -s ' ' — сжать множественные пробелы
tr -s '\n' — удалить пустые строки
tr -d '0-9' — удалить все цифры
tr ' ' '\n' — заменить пробелы на переводы строк
tr '\n' ' ' — заменить переводы строк на пробелы

Специальные символы:
\n — перевод строки
\t — табуляция
\r — возврат каретки
\b — backspace
\f — перевод формата
\v — вертикальная табуляция
\\ — обратный слеш
\0 — нулевой символ
\NNN — символ с восьмеричным кодом NNN
\xHH — символ с шестнадцатеричным кодом HH

Управление диапазонами:
tr 'A-Z' 'N-ZA-M' — ROT13 шифрование
tr 'a-zA-Z' 'A-Za-z' — инвертировать регистр
tr -d '[:cntrl:]' — удалить управляющие символы

Примеры с файлами:
cat file.txt | tr 'a-z' 'A-Z' — преобразовать файл в верхний регистр
tr -d '\r' < file.txt — удалить CR (Windows переводы строк)
tr -s '[:space:]' < file.txt — нормализовать пробелы
