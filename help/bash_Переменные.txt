Переменные Bash

Специальные переменные:
$0 — имя скрипта
$1, $2, ... — аргументы командной строки
$# — количество аргументов
$* — все аргументы как одна строка
$@ — все аргументы как отдельные строки
$? — код возврата последней команды
$$ — PID текущего процесса
$! — PID последнего фонового процесса

Предопределённые переменные:
$RANDOM — случайное число 0-32767
$SECONDS — время работы скрипта в секундах
$LINENO — текущий номер строки в скрипте
$PWD — текущая директория
$OLDPWD — предыдущая директория
$HOME — домашняя директория
$USER — имя пользователя
$UID — числовой ID пользователя
$SHELL — текущий шелл
$PATH — пути поиска команд
$PS1 — приглашение командной строки

Переменные среды:
$LANG — текущая локаль
$TERM — тип терминала
$EDITOR — редактор по умолчанию
$HOSTNAME — имя хоста
$HOSTTYPE — тип процессора
$OSTYPE — тип ОС

Bash-специфичные:
$BASH — путь к Bash
$BASH_VERSION — версия Bash
$BASH_VERSINFO — массив с информацией о версии
$BASHPID — PID текущего процесса Bash
$BASH_SUBSHELL — уровень подпроцесса
$SHLVL — уровень вложенности шелла

Опции шелла:
$- — текущие опции шелла
$FUNCNAME — имя текущей функции
$GROUPS — группы текущего пользователя

Массивы:
${array[@]} — все элементы массива
${!array[@]} — все индексы массива
${#array[@]} — количество элементов

Строковые переменные:
${#var} — длина строки
${var:pos} — подстрока с позиции pos
${var:pos:len} — подстрока длины len
${var/pattern/repl} — замена первого вхождения
${var//pattern/repl} — замена всех вхождений
${var/#pattern/repl} — замена в начале
${var/%pattern/repl} — замена в конце
