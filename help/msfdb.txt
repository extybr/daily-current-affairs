Инструкция по msfdb (база данных Metasploit Framework)

Инициализация:
msfdb init — инициализировать базу данных
msfdb reinit — переинициализировать базу
msfdb delete — удалить базу данных

Управление базой:
msfdb start — запустить базу данных
msfdb stop — остановить базу данных
msfdb restart — перезапустить базу
msfdb status — статус базы данных

Информация:
msfdb run — запустить базу и подключить msfconsole
msfdb --help — справка по командам

Конфигурация:
msfdb config — показать конфигурацию подключения
msfdb change-password — изменить пароль базы данных

Экспорт/Импорт:
msfdb export — экспортировать данные (устарело, используйте в msfconsole)
msfdb import — импортировать данные (устарело, используйте в msfconsole)

Управление службами:
msfdb enable — включить автозапуск службы
msfdb disable — выключить автозапуск

Проверка:
msfdb check — проверить подключение к базе
msfdb version — версия утилиты

Типичный рабочий процесс:
  1. msfdb init — создать БД
  2. msfdb start — запустить БД
  3. msfdb status — проверить статус
  4. msfdb run — запустить msfconsole с подключением
