Инструкция по printf

Основной синтаксис:
printf формат аргументы — форматированный вывод

Спецификаторы формата:
%s — строка
%d — целое число (десятичное)
%i — целое число (синоним %d)
%u — беззнаковое целое
%f — число с плавающей точкой
%e — научная нотация
%g — %f или %e в зависимости от величины
%x — шестнадцатеричное (строчное)
%X — шестнадцатеричное (заглавное)
%o — восьмеричное
%c — символ
%% — символ процента

Модификаторы:
%10s — строка шириной 10 символов
%-10s — выравнивание по левому краю
%.3f — 3 знака после запятой
%04d — целое с ведущими нулями (4 цифры)
%+d — показывать знак числа
% d — пробел перед положительными числами

Escape-последовательности:
\n — новая строка
\t — табуляция
\r — возврат каретки
\b — backspace
\a — звуковой сигнал
\\ — обратная косая черта
\0NNN — восьмеричный код символа
\xHH — шестнадцатеричный код символа

Примеры:
printf "%s\n" "Hello" — строка с переводом строки
printf "%d\n" 42 — целое число
printf "%f\n" 3.14159 — число с плавающей точкой
printf "%.2f\n" 3.14159 — два знака после запятой
printf "%10s\n" "text" — выравнивание по ширине 10
printf "%-10s %5d\n" "item" 100 — табличный вывод
printf "%04d\n" 42 — 0042
printf "%x\n" 255 — ff (шестнадцатеричное)
printf "%o\n" 64 — 100 (восьмеричное)
printf "%c\n" 65 — A (символ по коду)
printf "Progress: [%50s]\r" " — прогресс-бар

Множественные аргументы:
printf "%s %d %f\n" "Text" 42 3.14 — несколько аргументов

Массивы:
printf "%s\n" "${array[@]}" — каждый элемент с новой строки

Цвета:
printf "\033[31m%s\033[0m\n" "Red text" — красный текст

Даты и время:
printf "%(%Y-%m-%d)T\n" — текущая дата (в bash)

Форматирование чисел:
printf "%'d\n" 1000000 — 1,000,000 (с разделителем)

Без новой строки:
printf "%s" "text" — без \n в конце

Примеры таблиц:
    printf "%-15s %10s %10s\n" "Name" "Price" "Quantity"
    printf "%-15s %10.2f %10d\n" "Apple" 2.50 100
    printf "%-15s %10.2f %10d\n" "Banana" 1.75 200
    
Генерация строк:
printf "%0.s-" {1..50} — 50 дефисов

Форматированный вывод в файл:
printf "%s\n" "$data" > file.txt — запись в файл

Специальные форматы:
printf "%q\n" "string with spaces" — shell-экранированная строка
printf "%b\n" "text\nnewline" — интерпретация escape-последовательностей
