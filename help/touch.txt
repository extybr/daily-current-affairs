Инструкция по touch

Основные опции:
touch файл — создать пустой файл или обновить временные метки

Создание файлов:
touch file.txt — создать file.txt
touch file1 file2 file3 — создать несколько файлов
touch /путь/к/файлу — создать файл по полному пути

Изменение временных меток:
touch -a файл — изменить только время последнего доступа (atime)
touch -m файл — изменить только время модификации (mtime)
touch -c файл — не создавать файл если не существует
touch --no-create — то же что -c

Установка конкретного времени:
touch -t [[CC]YY]MMDDhhmm[.ss] файл — установить точное время
touch -d "строка_даты" файл — установить время из строки

Ссылки:
touch -h файл — работать с симлинками (не с целью)
touch --no-dereference — то же что -h

Примеры форматов времени:
touch -t 202312251200.00 — 25 декабря 2023, 12:00:00
touch -t 2312251200 — 25 декабря 2023, 12:00 (без секунд)
touch -d "2023-12-25 12:00:00" — ISO формат
touch -d "tomorrow" — завтра
touch -d "yesterday" — вчера
touch -d "next Monday" — следующий понедельник
touch -d "2 days ago" — 2 дня назад
touch -d "1 hour ago" — 1 час назад

Комбинированные опции:
touch -am файл — обновить и atime и mtime
touch -t 202312251200.00 -c файл — установить время если файл существует

Примеры:
touch newfile.txt — создать/обновить файл
touch -c missing.txt — ничего не делать если файла нет
touch -a log.txt — обновить время доступа
touch -m data.txt — обновить время модификации
touch -t 202301011200.00 timestamp.txt — установить конкретное время
touch -d "2 days ago" old.txt — установить время 2 дня назад
touch file{1..10}.txt — создать 10 файлов
touch .hidden — создать скрытый файл

Практическое применение:
touch /var/log/app.log — создать лог-файл если его нет
touch -m *.php — обновить время модификации всех PHP файлов
touch -d "$(date -d '1 year ago')" archive.txt — установить время год назад
touch -r source.txt target.txt — скопировать время из source в target

Копирование временных меток:
touch -r файл-образец целевой-файл — скопировать время из образца
